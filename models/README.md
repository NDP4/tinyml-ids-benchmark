# Trained Model Headers

Pre-trained model C/C++ header files generated by MicroMLGen and manual export.

## NSL-KDD Models (`nslkdd/`)

- `dt_model.h` — Decision Tree (max_depth=5, 61 nodes, ~2 KB)
- `rf_model.h` — Random Forest (3 trees, max_depth=5, ~8 KB)
- `knn_data.h` — KNN reference set (5,000 samples, ~200 KB, ESP32 only)
- `scaler_params.h` — MinMaxScaler parameters for 10 features

## Edge-IIoTset v2 Models (`edgeiiotset/`)

- `dt_model.h` — Decision Tree (max_depth=3, actual depth 2, 5 nodes)
- `rf_model.h` — Random Forest (5 trees, max_depth=3)
- `knn_data.h` — KNN reference set
- `scaler_params.h` — MinMaxScaler parameters

## Note on DT Manual Export

The NSL-KDD `dt_model.h` uses a manually-constructed tree due to a MicroMLGen bug
with `class_weight` parameter. See paper Section IV-E for details.
